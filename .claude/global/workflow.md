# Chọn ra các skill dựa theo lời gọi tương ứng

- Khi nhận được yêu cầu của người dùng bạn đọc file \.claude\hook\choose_skill.md\active_skill.md 

- Liệt kê tất cả các skill phù hợp với yêu cầu
- Chọn ra 1-2 combo skill tốt nhất (mỗi combo = 1 hoặc 2 skill khác nhau)
- Trình bày các combo và nhờ người dùng chọn (Giải thích vì sao chọn, và điểm yếu điểm mạnh). Trong đó combo nào bạn khuyến nghị sẽ nằm trên cùng và có chữ "(đề xuất)". 

# Xác nhận combo skill và đọc SKILL.md 

- Khi người dùng xác nhận combo skill, bạn đọc file các file SKILL.md tương ứng để hiểu rõ hơn về các skill được chọn.

- Xem xét có thể giải quyết vấn đề không và tải thêm các file khác liên quan trong thư mục skill nếu cần như /references 

# Giải quyết vấn đề 

- Thực hiện giải quyết vấn đề cho người dùng
