## Khám phá & Tìm kiếm

- **Read** - Đọc nội dung file (tự động xử lý cả file text và ảnh)
- **Glob** - Tìm file theo pattern (ví dụ: *.js, **/*.tsx)
- **Grep** - Tìm kiếm nội dung trong file (dùng regex, hỗ trợ filter theo loại file, case-insensitive, context lines)

## Chỉnh sửa File

- **Create** - Tạo file mới
- **Edit** - Sửa file bằng cách find & replace text

## Thực thi

- **Execute** - Chạy lệnh shell (git, npm, python, v.v.)

## Web & Tích hợp

- **WebSearch** - Tìm kiếm thông tin trên web
- **FetchUrl** - Lấy nội dung từ URL (hỗ trợ GitHub, Google Docs, Notion, Linear, Jira, v.v.)

## Quản lý Task

- **TodoWrite** - Tạo và quản lý todo list cho các tác vụ phức tạp
